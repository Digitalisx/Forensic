##gmail_parse.py 기본 정보
* Gmail로 수신한 메일의 링크를 파싱하여 해당 링크 안의 이미지 정보 값 추출 및 분석

* result.db : CSV의 내용을 DB화

* Final_Trave_Path : 전체 일자를 종합하여 이동 경로 표시
