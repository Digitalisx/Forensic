0. Pytsk3, Psutil 모듈의 설치가 필요합니다. (Python 2.7)

Pytsk3 : https://github.com/log2timeline/l2tbinaries/blob/master/win32/pytsk3-20160721.1.win32-py2.7.msi
Psutil : https://github.com/log2timeline/l2tbinaries/blob/master/win32/psutil-5.0.0.win32-py2.7.msi 

1. 관리자 권한으로 콘솔에서 ntfs.py를 실행 합니다.

python ntfs.py

2. 분석을 원하는 볼륨의 라벨을 입력합니다.

(ex) C:

3. 디렉터리를 입력합니다.

(ex) /Users/digit/Desktop/Forensic 스크린샷 파일 참조

4. 자동으로 분석을 진행하고 하위 폴더와 파일들을 출력해줍니다.

5. ntfs.py가 존재하는 디렉터리 아래에 HTML 파일이 생성됩니다. (JS 파일을 삭제하지 마세요)

6. HTML 파일을 조회할시 원하는 정보를 조회하고 정렬 할 수 있습니다.
